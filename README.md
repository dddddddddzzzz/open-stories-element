# `<story-view>`

Include `StoryViewElement`.

```html
<script src="https://unpkg.com/story-view-element" type="module" defer></script>
```

Render `<story-view>`.

```html
<story-view src="./feed.json" ttl="86400"></story-view>
```

## Attributes

- `ttl`
- `src`
